# This project is a pizza ordering web application that provides users with a convenient and interactive way to browse, select and purchase food online.
It combines an intuitive interface with dynamic functionality to deliver an engaging experience similar to modern food delivery platforms.

While working on this project, I learned and successfully implemented the following:

1. Developed a Single Page Application with multiple views using React Router, including: a homepage with a promotional carousel; categorized and individual product pages with detailed information; checkout page with a validated form, including delivery date and time selection.
2. Simulated API interaction by implementing a 3-second delay for product loading and displaying a loader to represent asynchronous data fetching.
3. Implementation of favorites and cart functionality, allowing users to save items and manage their orders efficiently.
4. Integration of promo code (SUMMER5 / MINUS10) functionality and automatic free delivery for orders over $25.
5. Implementation of a real-time search feature with smart debouncing (0.3s) for instant product filtering.
6. Responsive design ensuring optimal usability across various screen sizes.
7. Focus on clean UI/UX to simplify navigation and enhance the overall user experience.

## Installation
1. Clone the repo
```sh
git clone https://github.com/Koteykee/ReactPizza.git
```
2. Install NPM packages
```sh
npm install
```
3. Run application
```sh
npm run dev
```
Done! 😄
